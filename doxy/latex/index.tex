\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Main Goal}{Main Goal}}\label{index_intro_sec}
Password manager is used for storing and managing your passwords.~\newline
All files which store information about our passwords are encrypted.~\newline
While executing our program user are given that functionality\+:~\newline
~\newline

\begin{DoxyItemize}
\item Search passwords -\/ returns passwords that contain specific parameters.~\newline

\item Sort passwords -\/ returns a sorted list of all passwords. It should allow sorting by at least 2 different parameters at the same time, such as by name and category.~\newline

\item Add password -\/ adds a new password to the encrypted file. The user should be given the option to enter their own password and informed about how secure the password is and whether it has been used before. Additionally, an automatically generated password should be proposed to the user, giving them the option to choose certain parameters such as\+:~\newline
 Number of characters;~\newline
Whether to include uppercase and lowercase letters;~\newline
Whether to include special characters.~\newline

\item Edit password -\/ allows editing of data in an existing password.~\newline

\item Delete password(s) -\/ deletes the selected password or passwords. Before each deletion, we should notify the user, especially if more than one password is being deleted.~\newline

\item Add category -\/ adds a new category that we can use when creating new passwords.~\newline

\item Delete category -\/ deletes the category along with all the passwords assigned to that category.~\newline

\end{DoxyItemize}
