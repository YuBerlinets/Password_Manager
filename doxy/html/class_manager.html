◆ loadDataFromFile()

void Manager::loadDataFromFile ( )

Loading data from file to map.

Reads all data from decrypted json file, using nlohmann library and saves it to map "data", which represents such structure:
std::map<std::string, std::map<std::string, std::vector<std::string>>> data
<category, <website, <login, password>>>

  • map with "Category" as key, and map "CategoryValue" as value
  • map with "Website" as key and vector of string as value
  • vector where only two elements: login and password.
